# initialize
Initialize a new repo.

## Installation
````
npm i -g initialize
````

## Usage
````
  Usage: initialize <dirname>

  Options:

    -h, --help                    output usage information
    -V, --version                 output the version number
    -k, --keyword <tag>           add a keyword to package.json
    -d, --description <sentence>  add a description
````
